# Data and Model Policy

This repository may include compact poisoned CNN checkpoints and experiment outputs.

Full base LLM weights such as Llama-3.1-8B and Qwen3-8B are not stored in this GitHub repository. LLM experiments are reproduced from:
- public base model path configured locally;
- attack artifact JSON files;
- payload and target-layer metadata;
- defense scripts;
- result JSON/CSV files.

Large binary files should be tracked through Git LFS.
